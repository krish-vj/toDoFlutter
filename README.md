1.  Clone the repository:
    ```bash
    git clone https://github.com/krish-vj/toDoFlutter.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd your-repo-name
    ```
3.  Get the dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the application:
    ```bash
    flutter run
    ```
